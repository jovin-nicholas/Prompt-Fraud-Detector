# Prompt-Fraud-Detector
Evaluating Prompt Stealing Attacks in AI-Driven Assessment Content Generation

A lightweight toolkit to evaluate prompt stealing attacks in AI-generated assessment content.

## Contents
Dataset: A small collection of prompt-assessment pairs for testing and evaluation.

Colab Notebook: Step-by-step code to demonstrate prompt reconstruction and modifier classification.

## Getting Started
Open the Colab notebook to run experiments and explore the dataset.

Use the provided scripts to test prompt similarity analysis.
